O que é propriedade Display em CSS

A propriedade display em CSS controla como um elemento HTML é renderizado na página...

Display Características
Block Renderiza o elemento como um bloco, ocupando toda a largura disponível.
Inline O elemento é exibido na mesma linha e só ocupa o espaço necessário.
Inline-block Combina características de inline e block.
Flex Define um container flexível, útil para layouts responsivos.
